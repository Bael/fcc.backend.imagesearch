User stories

  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string. /api/search/:search
  2. I can paginate through the responses by adding a offset parameter to the URL. Offset must be greater than zero. For example /api/search/:search?offset=2
  3. I can get a list of the most recently submitted search strings with GET request for api/latestsearch/ url. Click on this link to see results
